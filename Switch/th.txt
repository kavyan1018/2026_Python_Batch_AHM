switch case -> c / c++ / java / c# / javascript 
    -> from the multiple options, select the one that matches the value of the variable


switch (experience){
    case 0:
        print("No experience")
        break
    case 1:
        print("1 year of experience")
        break
    case 2:
        print("2 years of experience")
        break
    default:
        print("More than 2 years of experience")
}
